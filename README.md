# Travilie
## Introduction

- Integrated multiple APIs, including Google Maps and weather APIs, to offer real-time information on restaurants,
hotels, and weather forecasts.

-  Developed a dynamic web application using JavaScript, HTML, and CSS to provide personalized travel
recommendations.



[RapidAPI](https://rapidapi.com/hub?utm_source=youtube.com/JavaScriptMastery&utm_medium=DevRel&utm_campaign=DevRel)
[Travel Advisor API](https://rapidapi.com/apidojo/api/travel-advisor?utm_source=youtube.com/JavaScriptMastery&utm_medium=DevRel&utm_campaign=DevRel)
[Open Weather Map API](https://rapidapi.com/community/api/open-weather-map?utm_source=youtube.com/JavaScriptMastery&utm_medium=DevRel&utm_campaign=DevRel)

Setup: run ```npm i && npm start``` to start the development server
